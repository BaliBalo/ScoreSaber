Usage

Add the page to OBS as a browser source.
Recommended size for the browser source: 1920x1080

Options

Query string options (after ? and separated by &)
  • u: your scoresaber ID (number after /u/ in your scoresaber url) - adds your profile picture, PP and rank in the top left corner while not in a song
  • hideSongInfo: hides the song info from the top left during a song
  • hideScoreInfo: hides the current score info (bottom bar) during a song
  • ip / port: override default websocket connection settings (useful for 2 PC setups)

Examples

  • overlay?u=2429129807113296: standard overlay showing Taichidesu's rank between songs
  • overlay?u=2429129807113296&hideSongInfo&hideScoreInfo: overlay showing only Taichidesu's rank between songs, and nothing during songs
  • overlay?hideSongInfo: overlay not showing anything in the top left corner, only current score info during songs
